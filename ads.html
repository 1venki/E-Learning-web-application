Course Name: Advanced data structures and algorithms

Description
Advanced data structures refer to complex and specialized data structures that are designed to efficiently solve specific types of problems. These structures often require more sophisticated algorithms for insertion, deletion, searching, and traversal compared to basic data structures like arrays, linked lists, stacks, and queues.

Ratings:  4.8/5

What you will learn in this course?

  • Understanding and implementing binary search trees (BSTs), AVL trees, red-black trees, and other self-balancing trees.
  • Analyzing the time complexities of tree operations like insertion, deletion, searching, and traversal.
  • Applications of trees in data storage, indexing, and searching algorithms.
  • Representing graphs using adjacency matrices, adjacency lists, and other data structures.
  • Exploring hashing techniques like hash tables, bloom filters, and hash functions.
